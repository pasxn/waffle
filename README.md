## waffle

##### A simple machine learning inference framework for the edge

### Installation

```bash
git clone https://github.com/pasxn/waffle.git
cd waffle
```
```bash
python -m pip install -e .
```
or if you wanna run tests,
```bash
python3 -m pip install -e '.[testing]'
```

### Running tests

```bash
python -m pytest -s -v
```
